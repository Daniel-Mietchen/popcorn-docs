Effects

Effects are transitions and visual effects composed on a track events start and end. This means that you can create visual cues to signify when an event is coming in or going out, notifying the user that an event has been fired. Effects are not limited to this tho, as in its simplest form, it is an event that is being fired when a Popcorn event starts and ends, allowing the user to do something at this time.
