Media Methods

Media methods are group of methods that pertain to various functions of the media element. These are functions like play, pause, set/get the current time, and so that make it easy to manipulate the media. This allows the user to programmatically control their media as well as perform various actions at times throughout the video, like call a function at 5 seconds.
