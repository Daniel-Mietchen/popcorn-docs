# media #

## Purpose ##

Instance property that contains a reference to the [HTMLVideoElement](https://developer.mozilla.org/en/DOM/HTMLVideoElement) or [HTMLAudioElement node](https://developer.mozilla.org/en/DOM/HTMLAudioElement).
